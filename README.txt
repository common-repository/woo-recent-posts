Displays a list of recent posts using thumbnails in your woo theme. Filter by category, set the number to display, and set the woo thumbnail size.

== Description ==

This plugin creates a re-usable widget that shows a list of recent posts in your sidebar. It is made specifically for the WooThemes framework, and takes advantage of the woothemes thumbnails.

= Features =
* Re-usable widget
* Filter by category
* Set the number of posts to display
* Set the thumbnail size
* Works with any WooTheme
* Plays well with other Plugins


== Installation ==

1. Upload the `woo-recent-posts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The "Woo - Recent Posts" widget should be available in the Widget area.
